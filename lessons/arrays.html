4.1 Definition and Characteristics

What is an Array?

An array is a collection of elements of the same data type stored in contiguous memory locations. Each element can be accessed using an index.

Key Characteristics:

  • Fixed size (size must be known at compile time)
  • Homogeneous data (all elements of same type)
  • Contiguous memory allocation
  • Zero-based indexing (first element at index 0)
  • Random access (O(1) time complexity)

Array Declaration and Initialization
/* Array Declaration and Initialization */
#include <stdio.h>

int main() {
    /* Method 1: Declaration with size */
    int arr1[5];
    
    /* Method 2: Declaration with initialization */
    int arr2[5] = {10, 20, 30, 40, 50};
    
    /* Method 3: Partial initialization (rest are 0) */
    int arr3[5] = {1, 2};  /* arr3 = {1, 2, 0, 0, 0} */
    
    /* Method 4: Size inferred from initialization */
    int arr4[] = {100, 200, 300};  /* Size = 3 */
    
    /* Accessing elements */
    printf("arr2[0] = %d\n", arr2[0]);
    printf("arr2[2] = %d\n", arr2[2]);
    
    /* Modifying elements */
    arr2[1] = 25;
    printf("Modified arr2[1] = %d\n", arr2[1]);
    
    return 0;
}

Memory Layout

  • If int arr[5] starts at address 1000
  • arr[0] is at address 1000
  • arr[1] is at address 1004 (assuming int = 4 bytes)
  • arr[2] is at address 1008
  • Formula: Address of arr[i] = base_address + (i * sizeof(datatype))

4.2 Array Operations

Traversal

Traversal means visiting each element of the array exactly once. Time Complexity: O(n)

Array Traversal
/* Traversing an Array */
#include <stdio.h>

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int n = 5;
    int i;
    
    printf("Array elements:\n");
    for(i = 0; i < n; i++) {
        printf("arr[%d] = %d\n", i, arr[i]);
    }
    
    /* Calculate sum */
    int sum = 0;
    for(i = 0; i < n; i++) {
        sum += arr[i];
    }
    printf("\nSum = %d\n", sum);
    
    /* Find maximum */
    int max = arr[0];
    for(i = 1; i < n; i++) {
        if(arr[i] > max) {
            max = arr[i];
        }
    }
    printf("Maximum = %d\n", max);
    
    return 0;
}

Insertion

Inserting an element into an array requires shifting elements to make space.

Time Complexity:

  • Insert at end: O(1)
  • Insert at beginning: O(n)
  • Insert at position i: O(n)
Insert Element at Position
/* Insert element at given position */
#include <stdio.h>

void insertElement(int arr[], int *n, int pos, int value) {
    int i;
    
    /* Shift elements to the right */
    for(i = *n; i > pos; i--) {
        arr[i] = arr[i - 1];
    }
    
    /* Insert the new element */
    arr[pos] = value;
    (*n)++;
}

void displayArray(int arr[], int n) {
    int i;
    printf("Array: ");
    for(i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
}

int main() {
    int arr[10] = {10, 20, 30, 40, 50};
    int n = 5;
    
    printf("Original ");
    displayArray(arr, n);
    
    /* Insert 25 at position 2 */
    insertElement(arr, &n, 2, 25);
    printf("After inserting 25 at position 2: ");
    displayArray(arr, n);
    
    /* Insert at beginning */
    insertElement(arr, &n, 0, 5);
    printf("After inserting 5 at beginning: ");
    displayArray(arr, n);
    
    /* Insert at end */
    insertElement(arr, &n, n, 60);
    printf("After inserting 60 at end: ");
    displayArray(arr, n);
    
    return 0;
}

Deletion

Deleting an element requires shifting elements to fill the gap.

Time Complexity:

  • Delete from end: O(1)
  • Delete from beginning: O(n)
  • Delete from position i: O(n)
Delete Element from Position
/* Delete element from given position */
#include <stdio.h>

void deleteElement(int arr[], int *n, int pos) {
    int i;
    
    if(pos < 0 || pos >= *n) {
        printf("Invalid position!\n");
        return;
    }
    
    /* Shift elements to the left */
    for(i = pos; i < *n - 1; i++) {
        arr[i] = arr[i + 1];
    }
    
    (*n)--;
}

void displayArray(int arr[], int n) {
    int i;
    printf("Array: ");
    for(i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
}

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int n = 5;
    
    printf("Original ");
    displayArray(arr, n);
    
    /* Delete element at position 2 */
    deleteElement(arr, &n, 2);
    printf("After deleting position 2: ");
    displayArray(arr, n);
    
    /* Delete first element */
    deleteElement(arr, &n, 0);
    printf("After deleting first element: ");
    displayArray(arr, n);
    
    return 0;
}

Searching

Linear Search: Sequential search through array elements.

Linear Search
/* Linear Search - O(n) */
#include <stdio.h>

int linearSearch(int arr[], int n, int key) {
    int i;
    
    for(i = 0; i < n; i++) {
        if(arr[i] == key) {
            return i;  /* Return index if found */
        }
    }
    
    return -1;  /* Return -1 if not found */
}

int main() {
    int arr[] = {64, 25, 12, 22, 11};
    int n = 5;
    int key = 22;
    int result;
    
    result = linearSearch(arr, n, key);
    
    if(result != -1) {
        printf("Element %d found at index %d\n", key, result);
    } else {
        printf("Element %d not found\n", key);
    }
    
    return 0;
}
Note: Binary Search (O(log n)) is more efficient but requires a sorted array. Linear Search works on both sorted and unsorted arrays.

Advantages

Benefits of Arrays

  • Fast Access: O(1) time to access any element using index
  • Simple Implementation: Easy to understand and implement
  • Memory Efficient: No overhead for storing links or pointers
  • Cache Friendly: Contiguous memory layout improves CPU cache performance
  • Predictable Memory: Memory allocation is straightforward and predictable
  • Multi-dimensional Support: Easy to create matrices and higher-dimensional structures
Example: Fast Random Access
/* Arrays provide O(1) access time */
#include <stdio.h>
#include <time.h>

int main() {
    int arr[1000000];
    int i;
    
    /* Initialize array */
    for(i = 0; i < 1000000; i++) {
        arr[i] = i;
    }
    
    /* Accessing first element - O(1) */
    printf("First element: %d\n", arr[0]);
    
    /* Accessing middle element - O(1) */
    printf("Middle element: %d\n", arr[500000]);
    
    /* Accessing last element - O(1) */
    printf("Last element: %d\n", arr[999999]);
    
    /* All accesses take same constant time! */
    
    return 0;
}

Disadvantages

Limitations of Arrays

  • Fixed Size: Cannot grow or shrink dynamically; size must be known at compile time
  • Costly Insertion/Deletion: Requires shifting elements, O(n) time complexity
  • Memory Waste: If array is not fully utilized, allocated memory is wasted
  • No Built-in Bounds Checking: Accessing out-of-bounds index causes undefined behavior
  • Contiguous Memory Required: Large arrays may fail to allocate if contiguous memory is unavailable
  • Homogeneous Data: Can only store elements of the same data type
Example: Costly Insertion
/* Insertion at beginning requires shifting all elements */
#include <stdio.h>

void insertAtBeginning(int arr[], int *n, int value) {
    int i;
    
    /* Shift all elements one position right - O(n) */
    for(i = *n; i > 0; i--) {
        arr[i] = arr[i - 1];
    }
    
    arr[0] = value;
    (*n)++;
}

int main() {
    int arr[100] = {10, 20, 30, 40, 50};
    int n = 5;
    int i;
    
    printf("Original array: ");
    for(i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
    
    /* Insert 5 at beginning - expensive operation */
    insertAtBeginning(arr, &n, 5);
    
    printf("After insertion: ");
    for(i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
    
    return 0;
}

When to Use Arrays

Arrays are ideal when:

  • The size of the data is known in advance
  • Random access to elements is required frequently
  • Memory efficiency is important (no pointer overhead)
  • You need to implement other data structures (stacks, queues, etc.)
  • Working with mathematical operations (matrices, vectors)

Consider alternatives when:

  • Size changes frequently (use dynamic structures like linked lists)
  • Frequent insertions/deletions at beginning or middle
  • Memory is limited and array might be underutilized
  • Need to store heterogeneous data types
Additional Array Operations

Reversing an Array

Reverse Array
/* Reverse an array */
#include <stdio.h>

void reverseArray(int arr[], int n) {
    int start = 0;
    int end = n - 1;
    int temp;
    
    while(start < end) {
        /* Swap elements */
        temp = arr[start];
        arr[start] = arr[end];
        arr[end] = temp;
        
        start++;
        end--;
    }
}

void displayArray(int arr[], int n) {
    int i;
    for(i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
}

int main() {
    int arr[] = {1, 2, 3, 4, 5};
    int n = 5;
    
    printf("Original array: ");
    displayArray(arr, n);
    
    reverseArray(arr, n);
    
    printf("Reversed array: ");
    displayArray(arr, n);
    
    return 0;
}

Rotating an Array

Rotate Array Left by K positions
/* Rotate array left by k positions */
#include <stdio.h>

void rotateLeft(int arr[], int n, int k) {
    int i, j;
    int temp;
    
    k = k % n;  /* Handle k > n */
    
    for(i = 0; i < k; i++) {
        temp = arr[0];
        
        /* Shift all elements left by 1 */
        for(j = 0; j < n - 1; j++) {
            arr[j] = arr[j + 1];
        }
        
        arr[n - 1] = temp;
    }
}

void displayArray(int arr[], int n) {
    int i;
    for(i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
}

int main() {
    int arr[] = {1, 2, 3, 4, 5};
    int n = 5;
    int k = 2;
    
    printf("Original array: ");
    displayArray(arr, n);
    
    rotateLeft(arr, n, k);
    
    printf("After rotating left by %d: ", k);
    displayArray(arr, n);
    
    return 0;
}

Merging Two Arrays

Merge Two Sorted Arrays
/* Merge two sorted arrays */
#include <stdio.h>

void mergeSortedArrays(int arr1[], int n1, int arr2[], int n2, int result[]) {
    int i = 0, j = 0, k = 0;
    
    /* Merge while both arrays have elements */
    while(i < n1 && j < n2) {
        if(arr1[i] <= arr2[j]) {
            result[k++] = arr1[i++];
        } else {
            result[k++] = arr2[j++];
        }
    }
    
    /* Copy remaining elements from arr1 */
    while(i < n1) {
        result[k++] = arr1[i++];
    }
    
    /* Copy remaining elements from arr2 */
    while(j < n2) {
        result[k++] = arr2[j++];
    }
}

void displayArray(int arr[], int n) {
    int i;
    for(i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
}

int main() {
    int arr1[] = {1, 3, 5, 7};
    int arr2[] = {2, 4, 6, 8};
    int n1 = 4, n2 = 4;
    int result[8];
    
    printf("Array 1: ");
    displayArray(arr1, n1);
    
    printf("Array 2: ");
    displayArray(arr2, n2);
    
    mergeSortedArrays(arr1, n1, arr2, n2, result);
    
    printf("Merged array: ");
    displayArray(result, n1 + n2);
    
    return 0;
}

Finding Duplicates

Find Duplicate Elements
/* Find and print duplicate elements */
#include <stdio.h>

void findDuplicates(int arr[], int n) {
    int i, j;
    int found = 0;
    
    printf("Duplicate elements: ");
    
    for(i = 0; i < n - 1; i++) {
        for(j = i + 1; j < n; j++) {
            if(arr[i] == arr[j]) {
                /* Check if already printed */
                int k, alreadyPrinted = 0;
                for(k = 0; k < i; k++) {
                    if(arr[k] == arr[i]) {
                        alreadyPrinted = 1;
                        break;
                    }
                }
                
                if(!alreadyPrinted) {
                    printf("%d ", arr[i]);
                    found = 1;
                }
                break;
            }
        }
    }
    
    if(!found) {
        printf("None");
    }
    printf("\n");
}

int main() {
    int arr[] = {1, 2, 3, 2, 4, 5, 3, 6};
    int n = 8;
    
    findDuplicates(arr, n);
    
    return 0;
}
Summary

Key Takeaways

  • Definition: Arrays are contiguous memory blocks storing homogeneous data
  • Access Time: O(1) for random access using index
  • Insertion: O(1) at end, O(n) at beginning or middle
  • Deletion: O(1) at end, O(n) at beginning or middle
  • Search: O(n) for unsorted, O(log n) for sorted (binary search)
  • Memory: Fixed size, contiguous allocation required
  • Best For: Fast random access, fixed-size collections
  • Worst For: Dynamic sizing, frequent insertions/deletions

Time Complexity Summary

Operation Best Case Average Case Worst Case
Access O(1) O(1) O(1)
Search O(1) O(n) O(n)
Insert at End O(1) O(1) O(1)
Insert at Position O(1) O(n) O(n)
Delete at End O(1) O(1) O(1)
Delete at Position O(1) O(n) O(n)
Remember: Arrays are the foundation for many other data structures like stacks, queues, and heaps. Understanding arrays thoroughly is essential for mastering more complex data structures!
