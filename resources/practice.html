üìä Array Problems

1. Find Maximum Element in Array
Easy Arrays
Write a function to find and return the maximum element in an array of integers.
Time Complexity: O(n)
Space Complexity: O(1)
Points: 10
Solve Show Hint Solution

üí° Hint

Initialize a variable with the first element, then iterate through the array comparing each element with your current maximum.

2. Reverse an Array
Easy Arrays
Implement a function to reverse an array in-place without using extra space.
Time Complexity: O(n)
Space Complexity: O(1)
Points: 10
Solve Show Hint Solution

üí° Hint

Use two pointers: one at the start and one at the end. Swap elements and move pointers towards the center.

3. Find Missing Number
Medium Arrays
Given an array containing n distinct numbers taken from 0, 1, 2, ..., n, find the one that is missing from the array.
Time Complexity: O(n)
Space Complexity: O(1)
Points: 20
Solve Show Hint Solution

üí° Hint

Use the mathematical formula: sum of n numbers = n*(n+1)/2. Subtract the sum of array elements from this.

4. Two Sum Problem
Medium Arrays
Given an array of integers and a target sum, find two numbers in the array that add up to the target. Return their indices.
Time Complexity: O(n)
Space Complexity: O(n)
Points: 20
Solve Show Hint Solution

üí° Hint

Use a hash table to store elements you've seen. For each element, check if (target - element) exists in the hash table.

5. Rotate Array by K Positions
Hard Arrays
Rotate an array to the right by k steps in-place with O(1) extra space.
Time Complexity: O(n)
Space Complexity: O(1)
Points: 30
Solve Show Hint Solution

üí° Hint

Use the reversal algorithm: reverse the entire array, then reverse the first k elements, then reverse the remaining elements.

üîó Linked List Problems

6. Insert Node at Beginning
Easy Linked Lists
Write a function to insert a new node at the beginning of a singly linked list.
Time Complexity: O(1)
Space Complexity: O(1)
Points: 10
Solve Show Hint Solution

üí° Hint

Create a new node, make its next pointer point to the current head, then update the head to point to the new node.

7. Detect Cycle in Linked List
Medium Linked Lists
Determine if a linked list has a cycle in it. Return true if there is a cycle, false otherwise.
Time Complexity: O(n)
Space Complexity: O(1)
Points: 20
Solve Show Hint Solution

üí° Hint

Use Floyd's Cycle Detection Algorithm (slow and fast pointers). If they meet, there's a cycle.

8. Reverse a Linked List
Medium Linked Lists
Reverse a singly linked list iteratively and return the new head.
Time Complexity: O(n)
Space Complexity: O(1)
Points: 20
Solve Show Hint Solution

üí° Hint

Use three pointers: previous, current, and next. Iterate through the list reversing the next pointers.

9. Find Middle Element
Medium Linked Lists
Find the middle node of a linked list. If there are two middle nodes, return the second middle node.
Time Complexity: O(n)
Space Complexity: O(1)
Points: 20
Solve Show Hint Solution

üí° Hint

Use two pointers: slow (moves 1 step) and fast (moves 2 steps). When fast reaches the end, slow will be at the middle.

10. Merge Two Sorted Lists
Hard Linked Lists
Merge two sorted linked lists and return it as a new sorted list.
Time Complexity: O(n+m)
Space Complexity: O(1)
Points: 30
Solve Show Hint Solution

üí° Hint

Use a dummy node and compare nodes from both lists one by one, appending the smaller one to the result list.

üìö Stack Problems

11. Valid Parentheses
Easy Stacks
Given a string containing just the characters '(', ')', '{', '}', '[' and ']', determine if the input string is valid.
Time Complexity: O(n)
Space Complexity: O(n)
Points: 15
Solve Show Hint Solution

üí° Hint

Push opening brackets onto the stack. When you encounter a closing bracket, check if it matches the top of the stack.

12. Evaluate Postfix Expression
Medium Stacks
Evaluate the value of an arithmetic expression in Reverse Polish Notation (postfix).
Time Complexity: O(n)
Space Complexity: O(n)
Points: 20
Solve Show Hint Solution

üí° Hint

Push operands onto the stack. When you encounter an operator, pop two operands, perform the operation, and push the result back.

13. Next Greater Element
Medium Stacks
For each element in an array, find the next greater element to its right. If it doesn't exist, return -1.
Time Complexity: O(n)
Space Complexity: O(n)
Points: 25
Solve Show Hint Solution

üí° Hint

Traverse the array from right to left. Use a stack to keep track of elements. Pop smaller elements before pushing current element.

14. Implement Queue using Stacks
Easy Queues
Implement a queue using two stacks. Support enqueue, dequeue, and peek operations.
Time Complexity: O(1) amortized
Space Complexity: O(n)
Points: 15
Solve Show Hint Solution

üí° Hint

Use one stack for enqueue and another for dequeue. Transfer elements between stacks when needed.

15. Generate Binary Numbers
Medium Queues
Generate binary numbers from 1 to n using a queue.
Time Complexity: O(n)
Space Complexity: O(n)
Points: 20
Solve Show Hint Solution

üí° Hint

Start with "1" in the queue. For each element, append "0" and "1" to generate new binary numbers.

üå≥ Tree Problems

16. Maximum Depth of Binary Tree
Easy Trees
Find the maximum depth (height) of a binary tree.
Time Complexity: O(n)
Space Complexity: O(h)
Points: 15
Solve Show Hint Solution

üí° Hint

Use recursion: depth = 1 + max(depth of left subtree, depth of right subtree). Base case: empty tree has depth 0.

17. Validate Binary Search Tree
Medium Trees
Determine if a binary tree is a valid binary search tree (BST).
Time Complexity: O(n)
Space Complexity: O(h)
Points: 25
Solve Show Hint Solution

üí° Hint

Use recursion with min and max boundaries. For each node, check if its value is within the valid range.

18. Level Order Traversal
Medium Trees
Print the level order traversal of a binary tree (breadth-first search).
Time Complexity: O(n)
Space Complexity: O(n)
Points: 20
Solve Show Hint Solution

üí° Hint

Use a queue. Start with the root, then for each node, enqueue its children and process them level by level.

19. Lowest Common Ancestor in BST
Hard Trees
Find the lowest common ancestor (LCA) of two nodes in a binary search tree.
Time Complexity: O(h)
Space Complexity: O(1)
Points: 30
Solve Show Hint Solution

üí° Hint

Use BST property: if both nodes are smaller, go left; if both are larger, go right; otherwise, current node is the LCA.

20. Serialize and Deserialize Binary Tree
Hard Trees
Design an algorithm to serialize a binary tree to a string and deserialize it back to the tree structure.
Time Complexity: O(n)
Space Complexity: O(n)
Points: 35
Solve Show Hint Solution

üí° Hint

Use preorder traversal for serialization. Use null markers for empty nodes. For deserialization, rebuild using the same order.

üí° Problem-Solving Tips

  1. Understand the Problem: Read carefully and identify input/output
  2. Think Before Coding: Sketch the approach on paper
  3. Start Simple: Solve for small inputs first
  4. Consider Edge Cases: Empty input, single element, duplicates
  5. Analyze Complexity: Think about time and space before implementing
  6. Test Thoroughly: Try different test cases
  7. Optimize Later: First make it work, then make it efficient
  8. Learn from Solutions: Study different approaches after solving
